# Penfold · Polymarket Global Dashboard

A single-file HTML dashboard of the most significant live Polymarket prediction markets — odds, outcomes, volume and end dates — curated by world significance (geopolitics, politics, economy, crypto, tech/AI, sports).

Odds are a snapshot from the public Polymarket Gamma API (2026-07-20). Open `index.html`, or view the live site via GitHub Pages.
